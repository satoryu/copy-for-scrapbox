# Contributing this project

There are several ways for you to contribute this project.

## Reporte bug you found and your idea

Reporing bugs you found is one of the best way to contribute. 
A bug report is just like a letter :love_letter: from those who I love, which cheers me. 

You have an idea to enhance this chrome extension? That's great!
Please tell me!
Anyway, please open new issue [here](https://github.com/satoryu/copy-for-scrapbox/issues/new/choose)

## Write codes to fix bug and develop new feature 

This project welcome any pull requests to help Scrapbox users. 

1. Fork this repository
2. Create your feature branch (eg. `git checkout -b my-new-feature`)
3. Commit your changes (eg. `git commit -am 'Add some feature'`)
4. Push to the branch (eg. `git push origin my-new-feature`)
5. Create new Pull Request
